# EssereEsteticaPrueba

Prueba de website con Codex para Essere Estetica.

Sitio web en React/Next.js para el emprendimiento familiar Essere Estetica.

## Archivos

- `src/app/page.tsx`: estructura principal del sitio.
- `src/components/booking-form.tsx`: formulario de solicitud de turnos.
- `src/app/globals.css`: estilos visuales y responsive.
- `public/images/hero-clinica.png`: imagen principal generada para el sitio.

## Desarrollo local

```powershell
npm install
npm run dev
```

Luego abrir `http://localhost:3000`.

## Build estatico

```powershell
npm run build
```

La salida exportada queda en `out/`.
